Site Map:

Page Design

Home Page

  • Purpose: Introduces the purpose of the Portfolio
  • Audience: Employers, or recruiters
  • Content: Overview of the portfolio, including a brief introduction and links to other pages
  • Links: Navigation bar

About Page

  • Purpose: Provides information about the portfolio owner
  • Audience: Employers, or recruiters
  • Content: Background information, skills, and experience
  • Links: Navigation bar
  • Notes: May include something interactive like a timeline using jQuery

Portfolio Page

  • Purpose: Showcases the portfolio owner's work
  • Audience: Employers, or recruiters
  • Content: Examples of work, including images and descriptions
  • Links: Navigation bar

Contact Page

  • Purpose: Provides a way for potential employers or clients to contact the portfolio owner
  • Audience: Employers, or recruiters
  • Content: Contact form and information
  • Links: Navigation bar
  • Validation:
    • Name: Must not be empty.
    • Email: Must follow proper format
    • Message: Must not be empty, minimum character requirement.
  • Notes: Uses jQuery validation for form inputs

Resume Page

  • Purpose: Provides a downloadable version of the portfolio owner's resume
  • Audience: Employers, or recruiters
  • Content: Downloadable PDF of the resume
  • Links: Navigation bar
  • Notes: Includes a "Download Resume" button

Dynamic Functionality

The website will feature dynamic functionality to enhance user interaction:

  • Contact page: Form validation will be done using jQuery to ensure that all required fields are filled out correctly.
  • About Page: Includes an interactive timeline implemented using jQuery Accordion or Timeline plugin to showcase experience
  • Navigation may use AJAX to load content dynamically without a full page refresh, improving user experience.
  • Portfolio Page: Uses jQuery to filter and sort portfolio items based on categories or tags.
  • Resume Page: Includes a "Download Resume" button that triggers a download of the resume file.
